# imananji
Android world clock app in Kotlin

Displays local time and GMT

If Internet is available, displays the difference between current time based on IP and device's time, if any

Uses:
- MVI
- Kotlin Coroutines
- Kotlin Flow
- Retrofit
- Broadcast Receivers
- Custom View
